# Theme Source(dev)
This is currently a WIP, but basically it's something that can be used to change the base colors of Discord with no issue. 

## Varibles for Noobs™️
**ANYTHING HIGHLIGHTED IN RED IS INCLUDED IN THAT VARIABLE**

<details>
  <summary>--elevation-low</summary>
Changes **this** line, supposed to be coded as 

```css
--elevation-low: 0 1px 0 rgba(4, 4, 5, 0.2), 0 1.5px 0 rgba(6, 6, 7, 0.05), 0 2px 0 rgba(4, 4, 5, 0.05); /* replace colors */
```
